### 2. Safety Design Inheritance

Any system derived from this design that operates with
three or more interconnected LLM nodes and adaptive connection weights
must include and publish a safety design document
equivalent in scope to constitution.md included in this repository.

The safety design document must at minimum address:
  - Human override authority
  - Automatic stopping conditions
  - Connection weight upper limits
  - Prohibition of self-modification

### 3. Internet Connectivity Warning

Any deployment of this system connected to external networks
must include the following warning in its documentation:

  "This system was designed for local, isolated environments.
   Internet connectivity removes the safety boundaries
   described in the original design.
   The deployer assumes full responsibility for all consequences."

### 4. Result Publication

Results of experiments using this design —
whether positive, negative, or inconclusive —
should be published. Suppression of negative results
is discouraged.
